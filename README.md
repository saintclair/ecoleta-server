#### Clone this repository
$ git clone https://github.com/saintclair/ecoleta-server.git

#### Go into the repository
$ cd ecoleta-server

#### Install dependencies
$ yarn install

#### Run Migrates
$ yarn knex:migrate

#### Run Seeds
$ yarn knex:seed

#### Start server
$ yarn dev

#### running on port 3333
